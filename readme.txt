WP2 Update is a modern solution for managing private WordPress theme and plugin updates directly from GitHub repositories. It leverages GitHub Apps for secure and automated workflows.

== Description ==

WP2 Update provides a seamless way to manage private WordPress updates. By using GitHub Apps, it ensures a secure and reliable connection between your WordPress site and your private repositories. Features include:

* Secure updates using GitHub Apps.
* Modern admin interface powered by Vite.
* Comprehensive logging and debugging tools.
* Support for both themes and plugins.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp2-update` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Follow the setup wizard to connect your GitHub App.

== Frequently Asked Questions ==

= How do I create a GitHub App? =

Follow the detailed guide in the plugin's README.md file or visit the GitHub wiki for step-by-step instructions.

= What permissions does the GitHub App require? =

The GitHub App requires read-only access to your repositories and metadata permissions.

= Can I use this plugin for free? =

Yes, WP2 Update is free and open-source. However, premium features may be introduced in the future.
